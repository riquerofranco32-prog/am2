01

Dominio & Curvas de Nivel

Funciones vectoriales reales, límites y continuidad en ℝⁿ

Función Vectorial Real

Asigna a cada vector del primer espacio un número real en el segundo. El dominio es el conjunto de vectores para los cuales existe un número real en el segundo espacio.

Dom f = { (x, y) ∈ ℝ² / condición }

Curvas de Nivel

Se obtienen al intersectar la superficie Z = f(x,y) con planos paralelos al plano XoY (Z = k) y proyectarlas sobre dicho plano.

Ck = { (x,y) ∈ Dom f / f(x,y) = k }

Límite de una función de dos variables

lim(x,y)→(x₀,y₀) f(x,y) = L
Simultáneo

Se evalúa directamente. Si existe, es el límite de la función.

Sucesivo

limx→x₀[limy→y₀ f(x,y)] = L. No se indetermina nunca.

Radial

y = m·x. Útil para demostrar que el límite no existe (si cambia con m).

Clasificación: Si el límite no existe → discontinuidad esencial. Si existe pero ≠ f(x₀,y₀) → discontinuidad evitable. Si existe e = f(x₀,y₀) → función continua.

Continuidad

Una función es continua en (a,b) si:

lim(x,y)→(a,b) f(x,y) = f(a,b)

También: si fxy(a,b) = fyx(a,b) → f es continua.

Plano Tangente — Función Afín

Plano que contiene las derivadas de las curvas originadas al cortar la superficie con planos paralelos a los ejes, en el punto P₀.

(Z − Z₀) = fx₀·(X−X₀) + fy₀·(Y−Y₀)
02

Derivadas Direccionales & Gradiente

Jacobiano, diferencial, máximo crecimiento

Derivada Total — Matriz Jacobiana

Se obtiene derivando cada función del segundo espacio respecto a cada variable del primero.

f'(X) = J = [ ∂f₁/∂x ∂f₁/∂y ] / [ ∂f₂/∂x ∂f₂/∂y ]

Diferencial de una Función

Aproximación al valor real con pequeño error que crece al alejarse del punto.

df = f'(X) · ΔX

Incremento Total

Indica qué tan cerca o lejos estamos del valor real de la función.

Δf = f(X₀ + ΔX) − f(X₀)

Derivada Direccional

Pendiente de la recta tangente a una curva generada al cortar la superficie con un plano que contenga al vector v̂ (versor unitario).

df/dv̂ = f'(X₀) · v̂ = ∇f(X₀) · v̂     donde v̂ = v / |v|
Procedimiento: Evaluar el Jacobiano de la función en el punto y multiplicar por el versor v̂.

⭐ Vector Gradiente — Dirección del Máximo Crecimiento

∇f(X₀) = f'(X₀) = ( fx(x,y) , fy(x,y) )
↑
Máximo crecimiento: dirección ∇f(X₀)
↓
Mínimo crecimiento: dirección −∇f(X₀)
+
Deriv. dir. máxima: |∇f(X₀)|
−
Deriv. dir. mínima: −|∇f(X₀)|
⊥
El gradiente es ortogonal a la curva de nivel en ese punto.
|∇f(X₀) · â| ≤ |∇f(X₀)| · |â| → df/dâ ≤ |∇f(X₀)|

Diferenciabilidad

f es diferenciable en X₀ si existe una función afín A(x) tal que f(X₀) = A(X₀) y [f(X₀) − A(X₀)] tiende a cero más rápido que (X − X₀).

Condición suficiente: Si las derivadas parciales fx y fy existen en un entorno de (a,b) y son continuas en (a,b) → f es diferenciable en (a,b).
03

Extremos Relativos & Plano Tangente

Puntos críticos, Hessiano, Taylor y Lagrange

Mínimo Local

f(x,y) alcanza un mínimo local en X₀ si:

∀ X ∈ ℝⁿ   f(X) ≥ f(X₀)

Máximo Local

f(x,y) alcanza un máximo local en X₀ si:

∀ X ∈ ℝⁿ   f(X) ≤ f(X₀)

Condición Necesaria — Existencia de Puntos Críticos

X₀ es un punto crítico si alguna de estas condiciones se cumple:

① ∇f(X₀) = 0̄
② fx(X₀) ∧ fy(X₀) no existen simultáneamente
Si f es diferenciable y ∇f(X) = 0̄ → todas las derivadas direccionales son cero → plano tangente horizontal en el PC.

Condición Suficiente — Determinante Hessiano Δ

Δ = fxx·fyy − (fxy)²
Δ > 0 y fxx > 0MÍNIMO relativo
Δ > 0 y fxx < 0MÁXIMO relativo
Δ < 0PUNTO SILLA (no es extremo)
Δ = 0No se puede asegurar nada

Polinomio de Taylor — Análisis del signo de d²f

f(X) ≈ f(X₀) + [fxΔX + fyΔY] + ½[fxx(ΔX)² + 2fxyΔXΔY + fyy(ΔY)²]
f(X) − A(X₀) = d²f(X₀)
+
d²f > 0 → MÍNIMO (f sobre el afín)
−
d²f < 0 → MÁXIMO (f bajo el afín)

⭐ Extremos Condicionados — Multiplicadores de Lagrange

Optimización con restricciones. Se construye la función auxiliar:

F(X) = f(x,y) + λ₁G₁ + λ₂G₂ + ... + λₙGₙ
1

Armar la función auxiliar F(X).

2

Derivar en fx y fy e igualar a cero.

3

Despejar λ e igualar las expresiones obtenidas.

4

Determinar X, Y, Z y reemplazar en la condición → obtener los PC.

5

Análisis de naturaleza: despejar variable de la condición, reemplazar en f(x) de una sola variable.

6

Derivar → f'(x) → f''(x): si f'' > 0 → mínimo; si f'' < 0 → máximo.

04

Funciones Paramétricas & Curvas

ℝ → ℝᵐ, trayectorias, velocidad y aceleración

Función Vectorial de Variable Real

Función cuyo dominio es ℝ y cuya imagen es un conjunto de vectores. El parámetro t indica tiempo → la curva indica una trayectoria.

g : ℝ → ℝᵐ    g(t) = ( g₁(t), g₂(t), ..., gₘ(t) )
Solo se grafica la imagen (curvas en plano o espacio, NO superficies).

Curvas en el Plano ℝ → ℝ²

x = g₁(t)
y = g₂(t) → g(t) = (g₁(t), g₂(t))

Circunferencia: g(t) = (r·cos t, r·sen t)
Elipse: x = a·cos t, y = b·sen t

Curvas en el Espacio ℝ → ℝ³

g(t) = ( g₁(t), g₂(t), g₃(t) )

El gráfico de la imagen representa una curva en el espacio xyz.

Límite y Continuidad

limt→a g(t) = ( lim g₁(t), lim g₂(t), ..., lim gₘ(t) )

Continua si: limt→a g(t) = g(a)

Derivada — Componente a componente

g'(t) = ( g₁'(t), g₂'(t), ..., gₘ'(t) )

g'(t) = vector tangente a la curva = vector velocidad V(t)

⭐ Recta Tangente a una Trayectoria

r(t) = g(t₀) + t · g'(t₀)
g(t₀) = vector posición (punto)
g'(t₀) = vector director (tangente / velocidad)

Velocidad y Aceleración Vectorial

v
g'(t) = V(t) → vector velocidad
|v|
|g'(t)| = v(t) → rapidez (velocidad escalar)
a
g''(t) = ā(t) → vector aceleración
|a|
|g''(t)| = a(t) → aceleración escalar

Componente Tangencial y Normal:

τ̂ = g'(t) / |g'(t)| = V(t) / v(t)
a_T = |v(t)|'   (componente escalar)
a_N = √[ a(t)² − a_T² ]
Con rapidez constante: V(t) ⊥ ā(t)

Longitud de Curva (Arco)

Lc = ∫ab |g'(t)| dt

Para una curva g(t) = (g₁(t), g₂(t), g₃(t)) continua por tramos en [a,b].

05

Campos Vectoriales & Integrales de Línea

Conservativos, potencial, trabajo, rotacional y divergencia

Campo Vectorial F : ℝⁿ → ℝⁿ

Representación gráfica mediante flechas: origen en punto del dominio, extremo en punto de la imagen.

F : ℝ² → ℝ²   F(x,y) = (M, N)
F : ℝ³ → ℝ³   F(x,y,z) = (M, N, P)

⭐ Campos Conservativos

Un campo F(x,y) = (M,N) es conservativo si las derivadas cruzadas son iguales:

∂N/∂x = ∂M/∂y
Definición alternativa: F es conservativo si existe f diferenciable tal que F = ∇f. Esta f se llama función potencial.
En ℝ³: se verifica calculando el rotacional. Si rot(F) = 0̄ → campo conservativo.

Función Potencial — Cálculo

M = ∂f/∂x → f = ∫M dx = f = ∫N dy ← N = ∂f/∂y
1

Integrar M respecto a x → obtener f(x,y) + φ(y)

2

Derivar respecto a y e igualar a N → despejar φ'(y)

3

Integrar φ'(y) → construir f(x,y) completa.

f(x,y) se forma reuniendo lo que se repite en ambas integrales, luego lo que no se repite.

Integral de Línea en un Campo Vectorial — Trabajo

Calcula el trabajo para desplazar una partícula por la curva γ de a hasta b en el campo F.

WT = ∫ab F[g(t)] · g'(t) dt
1. Parametrizar F(x,y,z)
2. Derivar g(t) → g'(t)
3. Componer F[g(t)]
4. Producto punto F[g(t)]·g'(t)
5. Integral definida
También: W = ∫γ M dx + N dy

Teorema Fundamental de las Integrales de Línea

En campo conservativo el trabajo no depende de la trayectoria:

WT = f[x(b), y(b)] − f[x(a), y(a)] = f(b) − f(a)
Si C es curva cerrada y el campo es conservativo → WT = 0

Rotacional (rot F)

rot(F) = ∇ × F
= (∂P/∂y−∂N/∂z)i − (∂P/∂x−∂M/∂z)j + (∂N/∂x−∂M/∂y)k
rot F = 0 → campo conservativo (en ℝ³)

Divergencia (div F)

div F(x,y) = ∂M/∂x + ∂N/∂y
div F(x,y,z) = ∂M/∂x + ∂N/∂y + ∂P/∂z

Función escalar (a diferencia del rotacional que es vectorial).

06

Teorema de Green

Relación entre integral de línea y rotacional sobre una región plana

⭐ Enunciado

Sea R una región del plano simple cuyo borde es una curva cerrada C, suave a trozos, orientada en sentido antihorario, y si M y N tienen derivadas parciales continuas en una región abierta que contiene a R, se cumple:

∮C M dx + N dy = ∬R (∂N/∂x − ∂M/∂y) dA
El término (∂N/∂x − ∂M/∂y) es el rotacional del campo F = (M,N).

Tres casos según el valor del Rotacional

ROT = 0
Campo conservativo → WT = 0
La integral de línea sobre cualquier curva cerrada es cero.
ROT = cte
WT = ROT · Área de la región
Se calcula el área (forma geométrica) y se multiplica.
ROT = f(x,y)
WT = ∬R f(x,y) dA
Se aplica integral doble sobre la región encerrada.

Rotacional en ℝ³

rot(F) = i(∂P/∂y − ∂N/∂z) − j(∂P/∂x − ∂M/∂z) + k(∂N/∂x − ∂M/∂y)
Si rot F = 0 en todo el espacio → F es conservativo si y solo si el dominio es todo ℝ³.

Interpretación

El Teorema de Green dice que la circulación antihoraria de un campo vectorial alrededor de una curva cerrada simple es igual a la integral doble del rotacional del campo sobre la región encerrada por la curva.

Resumen de gráficas: Funciones reales → superficies · Funciones paramétricas → curvas · Funciones de campo → flechas
07

Integrales Múltiples

Dobles, regiones, cálculo de áreas y volúmenes

Integral Doble f : ℝ² → ℝ

Se utiliza para calcular áreas o volúmenes. Se divide el dominio D en pequeños cuadrados ΔxΔy y se proyectan hacia la superficie.

V = ∬D f(x,y) dx dy    (D = dominio de integración)
∬D f(x,y) dxdy = limΔx→0, Δy→0 Σ f(xᵢ,yᵢ) ΔxᵢΔyᵢ

Regiones Rectangulares

Los límites son constantes. Se puede integrar en cualquier orden (Teorema de Fubini):

V = ∫ab ∫cd F(x,y) dy dx = ∫cd ∫ab F(x,y) dx dy

⭐ Regiones No Rectangulares — Teorema de Fubini

Tipo I · Verticalmente Simple

x entre constantes, y entre funciones

D₁ = { (x,y) / a ≤ x ≤ b, g₁(x) ≤ y ≤ g₂(x) }
V = ∫ab ∫g₁(x)g₂(x) f(x,y) dy dx
Tipo II · Horizontalmente Simple

y entre constantes, x entre funciones

D₂ = { (x,y) / c ≤ y ≤ d, h₁(y) ≤ x ≤ h₂(y) }
V = ∫cd ∫h₁(y)h₂(y) f(x,y) dx dy
Clave: al armar los límites, siempre entrar por una función y salir por otra. Se aplica trazando líneas horizontales o verticales sobre la región.

Cálculo de Área

Se considera f(x,y) = 1 → la integral doble calcula el área:

A = ∬D dx dy = ∫ab dx ∫g₁(x)g₂(x) dy

Cálculo de Volumen

Igual que el área pero con f(x,y) obtenida al despejar z de la superficie:

V = ∬D f(x,y) dA

Propiedades de la Integral Doble

∬ c·f dA = c·∬ f dA
∬ [f ± g] dA = ∬ f dA ± ∬ g dA
f ≥ 0 → ∬ f dA ≥ 0
f ≥ g → ∬ f dA ≥ ∬ g dA
∬R f dA = ∬R₁ f dA + ∬R₂ f dA   (suma de regiones)
